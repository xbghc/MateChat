# Runtime API Examples

本页演示了 VitePress 提供的一些运行时 API 的用法。

运行时API `useData()` 可用于访问当前页面的站点、主题和页面数据。它适用于 `.md` 和 `.vue` 文件：

```md
<script setup>
import { useData } from 'vitepress'

const { theme, page, frontmatter } = useData()
</script>

## Results

### Theme Data
<pre>{{ theme }}</pre>

### Page Data
<pre>{{ page }}</pre>

### Page Frontmatter
<pre>{{ frontmatter }}</pre>
```

<script setup>
